# planner task checklistitem list

Lists the checklist items of a Planner task.

## Usage

```sh
m365 planner task checklistitem list [options]
```

## Options

`-i, --taskId <taskId>`
: ID of the task

<Global />

## Examples

Lists the checklist items of a Planner task.

```sh
m365 planner task checklistitem list --taskId 'vzCcZoOv-U27PwydxHB8opcADJo-'
```
